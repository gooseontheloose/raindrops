# Raindrops Website

A sleek, VRChat-focused gaming community website that includes a ban appeal system and a custom live CMS (Content Management System) for easy editing. Built entirely with Vanilla JS, HTML, CSS, and basic PHP.

## Features
- Dynamic text updates via the custom Staff Dashboard CMS
- Interactive, responsive background animations and UI features
- Direct-to-Discord Ban Appeal System integration
- Highly portable with built-in persistent storage setup

## Deploying
This repository is configured to deploy instantly on **Railway**. 
Please refer to the detailed [RAILWAY_SETUP.md](RAILWAY_SETUP.md) for full instructions on how to get it running and retain data properly using Railway Volumes.

### Running it Locally
If you want to run this locally to make more CSS changes, simply start a basic PHP server in the root of the project:
```sh
php -S localhost:8000
```
Then navigate to `http://localhost:8000`.
